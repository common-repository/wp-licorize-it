== Description ==

[Licorize for WordPress](http://licorize.com/) is a wordpress plugin that allow you to add a "Add to Licorize" button to each blog post in both the contents of your post and the RSS feed.

== Installation ==

=How to use it :=

    * Download it
    * Extract all files from the zip archive
    * Copy the wp-licorize-it folder to wp-content/plugins
    * Activate the plugin through the "Plugins" menu in wordpress
    * Configure style options as desire
    * That's it !
